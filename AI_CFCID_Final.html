Project details

About this project

AI-CFCID is an AI-powered customer feedback classification and insights platform built for EM 680 at St. Cloud State University. It solves a real problem in e-commerce and retail: the inability to automatically classify, prioritize, and surface insights from large volumes of unstructured customer feedback.

The system uses zero-shot classification via the Claude API (Anthropic) — no training data required. Each feedback item is labeled with sentiment, operational category, and urgency level, along with AI-generated reasoning and a recommended operational action.

Setup Guide & Presentation Instructions

Step-by-step guide to share this dashboard with your team and professor via Google Drive

Before your presentation — 5 minute setup
1
Get your Anthropic API key
Go to console.anthropic.com → Sign in → Click API Keys in the left sidebar → Click Create Key → Copy the key (starts with sk-ant-).
Keep this key private — delete it from the console immediately after your presentation.
2
Add your API key to this file
Download this HTML file → Open it in Notepad (Windows) or TextEdit (Mac) → Press Ctrl+H (Find & Replace) → Find: YOUR_API_KEY_HERE → Replace with your actual key → Save the file.
3
Upload to Google Drive
Go to drive.google.com → Click + New → File upload → Select the saved HTML file → Wait for upload to complete.
4
Share with your team & professor
Right-click the file in Drive → Share → Click "Change to anyone with the link" → Set to Viewer → Copy link → Send to your team and professor.
They open the link → Google Drive shows a preview → they click Open or Download → open the file in their browser.
5
After your presentation — revoke the API key
Go back to console.anthropic.com → API Keys → click the Delete button next to your key. This instantly stops anyone from using it even if they saved the file. It takes 10 seconds and protects your account completely.
Important notes for your professor demo

✓ Overview dashboard & all charts
✓ Feedback Data table & search
✓ Demographics page
